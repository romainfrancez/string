# string

A set of functions to extend the capabilities of String.

## Getting started
### Browser
Download the file [string.min.js](js/string.min.js) and include it into your web pages.

### Node.js
```shell
npm install git+https://github.com/romainfrancez/string.git
```

### Documentation
```shell
npm install
grunt doc
```
